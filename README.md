# SponsorMatch BI

SponsorMatch BI is a creator analytics dashboard that helps influencers analyze sponsorship readiness, engagement, and revenue opportunities.

## Live Demo

https://sponsor-match-bi.vercel.app

## Features

- User authentication (Login / Signup / Logout)
- Protected dashboard
- Creator management (Create, Read, Update, Delete)
- Search, filtering, and sorting
- Pagination
- KPI cards
- Revenue tracking
- Charts and analytics
- CSV export
- Sidebar navigation
- User profile page

## Tech Stack

- Next.js
- React
- Supabase
- Tailwind CSS
- Chart.js
- Vercel
